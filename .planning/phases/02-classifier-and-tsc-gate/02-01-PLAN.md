    4. Create `_find_tsconfig(workspace: Path) -> Path` function:
       - Check for `workspace / "tsconfig.json"`, then `workspace / "tsconfig.base.json"`
       - Return first found, raise `FileNotFoundError` with message `f"No tsconfig.json found in {workspace}"` if neither exists
       - Google-style docstring

    5. Create `TscGate` class:
       - `__init__(self, engine: TsMorphProjectEngine, workspace: Path) -> None`:
         - Store `self._engine = engine`
         - Call `self._tsconfig_path = _find_tsconfig(workspace)` as a pre-flight check.
           This raises FileNotFoundError immediately if no tsconfig exists in the
           workspace, enforcing TSC-05 at gate construction time rather than deferring
           to first snapshot call. The stored `_tsconfig_path` is not yet passed to the
           engine (Phase 3's migration runner will construct the engine with it), but the
           pre-flight check ensures TSC-05 violations are caught loudly and early.
       - `async def snapshot_before(self) -> TscSnapshot`: call `await self._engine.get_diagnostics()`, filter to `d.severity == "error"`, build `error_fingerprints` as `[(d.file_path, d.line, d.code) for d in errors]`, return `TscSnapshot(error_count=len(errors), error_fingerprints=error_fingerprints)`. Store result in `self._before`.
       - `async def snapshot_after(self) -> TscSnapshot`: same as above but stores in `self._after`.
       - `def compare(self) -> TscGateResult`: compute `new = self._after.fingerprint_set - self._before.fingerprint_set`, return `TscGateResult(before=self._before, after=self._after, new_errors=sorted(new), passed=len(new) == 0)`. Raises `RuntimeError` if `_before` or `_after` not yet taken.
       - Google-style docstrings on class and all public methods.
