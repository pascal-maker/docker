       - If blocked lifecycle methods found: return `ComponentClassification(file_path=info.file_path, component_name=info.component_name, tier=MigrationTier.MANUAL, rationale=f"Hard-blocked: contains {', '.join(sorted(blocked))}")`

       - signals > 2: tier = MANUAL, rationale listing all signals
       - Return ComponentClassification with computed tier and rationale
       - Google-style docstring
