<objective>
Implement the PydanticAI classifier agent that assigns tier and rationale to components not resolved by the deterministic layer, and the top-level `classify_components` orchestrator that chains blocklist -> heuristics -> AI into a complete MigrationScope result.

Purpose: The AI classifier handles edge cases where deterministic rules are insufficient — components with ambiguous complexity signals that require judgment. The orchestrator function is the public API that Phase 3 (executor) will call to get a full workspace classification.

Output: `classifier.py` extended with AI agent factory and classify_components function, tests using PydanticAI TestModel.
